# 👕 Glowear - Custom T-Shirt Ordering System

Glowear adalah sistem informasi pemesanan kaos custom berbasis **Service-Oriented Architecture (SOA)** yang dikembangkan untuk membantu proses bisnis konveksi seperti Glomed&co.

Aplikasi ini terdiri dari:
- 🧠 Backend (API Service)
- 🖥️ Web Admin (CMS)
- 📱 Mobile App (User)

---

## 🚀 Tech Stack

### Backend
- NestJS
- Prisma ORM
- PostgreSQL

### Frontend (Admin Web)
- Next.js
- React
- Tailwind CSS

### Mobile App (User)
- React Native
- Expo
